Stanford CS242 assign4 - System F interpreter + typechecker (Lam1)
Built + run in WSL Ubuntu with OCaml 4.14.1, Core v0.16.

NOTE ON THE REFERENCE BINARY:
  The course ships a prebuilt reference.byte compiled for OCaml ~4.02
  (bytecode magic Caml1999X011). The only OCaml toolchain available on
  this machine is 4.14 (Caml1999X031), which refuses to run the older
  bytecode. Correctness is therefore verified against (a) the assignment's
  own inline_tests assertion suites (substitution / alpha-equivalence /
  de-Bruijn / typechecker / interpreter) which all pass, and (b) the known
  mathematical result of each example program (checked below).
